== Description ==

Add-On for MP Stacks which displays anything with an "Embed Code" in a grid. Each grid item can be a different embeded item (IE YouTube, Vimeo, Tweets, Instagrams, BandCamp, SoundCloud, etc).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the mp-stacks-embedsgrid folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Build Bricks under the Manage Stacks menu. 
4. Publish your bricks into a Stack.
5. Put Stacks on pages using the shortcode or the Add Stack button.
